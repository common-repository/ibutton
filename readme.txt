iButton is simple shortcode button. It has 15 different colours and 4 different sizes. You can use it easily by shortcode.

== Description ==
iButton in easy to use inverted shortcode button plugin. You can use the buttons by placing shortcode in your post or page. iButton is compatible with all themes, that mean you can use this button aside any theme. We tested with 20 themes with this plugin, everything worked without any problem.

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `ibutton` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[ibutton url=“http://google.com” text=“Click Here”]` in your post or page. Visit Here for more attribute list of the button shortcode: http://demo.solayman.net/ibutton/ibutton-documention/
